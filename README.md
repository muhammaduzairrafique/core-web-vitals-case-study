# Core Web Vitals Case Study

A WordPress speed optimization and Core Web Vitals case study showing how performance improvements can support better user experience and SEO.

## Focus Areas

- PageSpeed Insights review
- Core Web Vitals improvement
- LCP optimization
- INP/TBT improvement
- CLS stability
- Image optimization
- Caching setup
- CSS/JS optimization
- Database cleanup
- Mobile performance

## Example Improvements

- Mobile performance improved from 50 to 95
- Desktop performance improved from 35 to 98
- LCP improved significantly
- TBT reduced to 0ms
- CLS improved to a stable score

## Purpose

This repository documents speed optimization concepts and performance-focused SEO improvements.
